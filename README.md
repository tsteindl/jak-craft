# jak-craft

## Wie verwende ich Mod?
- https://github.com/tsteindl/jak-craft/releases/download/server-latest/jakcraft-server.jar herunterladen
- In Forge mods folder hinzufügen
- andere Mods herunterladen
  - The Dawn Era Legacy (DISCONTINUED). Version: 1.20.1-0.58.94
  - AstermirLib
  - Illage and Spillage Respillaged
  - Refined Bossbars
  - WorldEdit


## Installation (Development)
- Java 17 haben
- In IDE öffnen (Bsp anhand von IntelliJ)
- Gradle Build ausführen (Tab ganz rechts auf der Seite)
- Gradle Task "forgegradle runs/genIntellijRuns" ausführen
- Runkonfiguration "runClient" ausführen (ggfs. IDE neustarten wenn nicht vorhanden) (Wichtig: NICHT die gradle Task run<Thing> ausführen), es sollte sich ein MC-Fenster öffnen
- Programmieren, usw...
- Mod kompileren mittels ".\gradlew build" und resultierendes .jar in ".minecraft/mods" einfügen
- CurseForge starten und Profil starten
- Testen
